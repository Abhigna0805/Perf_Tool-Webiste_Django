Introduction to Industrial OS

Industrial OS is a Debian-based Linux distribution. Debian is a broadly utilized, enterprise-class, open source Linux operating system (OS) that allows different levels of OS customization for multiple architectures.
The majority of Industrial OS packages come unchanged from Debian. On top of the Debian base, Industrial OS provides additional components, such as Docker and SWUpdate, to help embedded platform developers assemble and configure the OS before deployment.
Industrial OS introduces the benefits of a customizable OS on top of a stable Linux distribution
